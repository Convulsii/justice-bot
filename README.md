**Мощный Discord бот с экономикой играми фермой рыбалкой и кучей всего**

## 📦 УСТАНОВКА

### 1. Клонируй репозиторий
```bash
git clone https://github.com/Justicemmm/JusticeBot.git
cd JusticeBot
```

### 2. Установи зависимости
```bash
pip install -r requirements.txt
```

### 3. Настрой токен и ID каналов
Открой `Justice.py` и вставь свои данные:

```python
TOKEN = "ТВОЙ_ТОКЕН"

WELCOME_CHANNEL_ID = 123456789      # Канал приветствий
LOGS_CHANNEL_ID = 123456789         # Канал логов
LEVEL_CHANNEL_ID = 123456789        # Канал уровней
TICKET_CATEGORY_ID = 123456789      # Категория для тикетов
TICKET_CREATE_CHANNEL_ID = 123456789 # Канал с кнопкой тикета
```

### 4. Запусти бота
```bash
python Justice.py
```

## ❗ ЧАСТЫЕ ОШИБКИ

| Ошибка | Решение |
|--------|---------|
| `ModuleNotFoundError: No module named 'openmeteo_requests'` | `pip install openmeteo-requests` |
| `discord.errors.LoginFailure` | Неправильный токен |
| `sqlite3.OperationalError: no such table` | База данных создастся сама при первом запуске |
| `IndentationError` | Проверь отступы в коде |
