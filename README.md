## 🚀 Быстрый старт

### 1. Клонирование репозитория
```bash
git clone https://github.com/ваш-ник/justice-bot.git
cd justice-bot
```

### 2. Установка зависимостей
```bash
pip install discord.py aiosqlite openai aiohttp
```

### 3. Настройка переменных окружения
Создайте файл `.env`:
```
DISCORD_TOKEN=ваш_токен_бота
AI_API_KEY=ваш_api_ключ
```

### 4. Запуск
```bash
python Justice.py
```

### 5. Приглашение бота на сервер
```
https://discord.com/oauth2/authorize?client_id=ВАШ_CLIENT_ID&permissions=8&scope=bot
```

## 🛠️ Для администраторов

### Как получить ID каналов/ролей:

1. **Включите Режим разработчика:**
   - Discord → Настройки → Дополнительно → Режим разработчика (ВКЛ)

2. **Копирование ID:**
   - **Канал:** ПКМ по каналу → Копировать ID
   - **Роль:** ПКМ по роли в списке участников → Копировать ID
   - **Сервер:** ПКМ по названию сервера → Копировать ID

### Где заменить ID в коде:

В файле `Justice.py` в разделе `# ========== КОНФИГУРАЦИЯ ==========`:

```python
# ID каналов (ОБЯЗАТЕЛЬНО ЗАМЕНИТЬ!)
WELCOME_CHANNEL_ID = 123456789012345678      # Канал приветствий
LOGS_CHANNEL_ID = 123456789012345678         # Канал логов
LEVEL_CHANNEL_ID = 123456789012345678        # Канал уровней
COLOR_ROLE_CHANNEL_ID = 123456789012345678   # Канал для цветных ролей
MUTED_ROLE_ID = 123456789012345678           # ID роли для мута
DEFAULT_ROLE_ID = 123456789012345678         # Роль при входе
VC_CREATE_CATEGORY_ID = 123456789012345678   # Категория для голосовых
VC_TRIGGER_CHANNEL_ID = 123456789012345678   # Триггерный канал

# Тикеты
TICKET_CATEGORY_ID = 123456789012345678      # Категория для тикетов
TICKET_CREATE_CHANNEL_ID = 123456789012345678 # Канал с кнопкой тикетов

# Роли поддержки (можно несколько)
SUPPORT_ROLE_IDS = [
    123456789012345678,  # Роль поддержки 1
    123456789012345678,  # Роль поддержки 2
]
```
