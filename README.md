## ✨ Особенности

| Категория | Возможности |
|-----------|-------------|
| 👤 **Профиль** | Уровни, опыт, статистика сообщений (день/неделя/месяц) |
| 💰 **Экономика** | Баланс, банк, переводы, ежедневные бонусы |
| 🎲 **Игры** | Казино, слоты, кубик, монетка, КНБ, блэкджек, покер, крестики-нолики |
| ⭐ **Репутация** | +/- репутация, система КД 1 час |
| 🔫 **Ограбление** | Шанс 5%, потеря репутации при провале |
| 🏆 **Топы** | По сообщениям, репутации, балансу, уровню |
| 🛍️ **Магазин** | Покупка наград, инвентарь |
| 🎭 **Ролевые** | 23+ команд с аниме-гифками |
| 🛠️ **Модерация** | Варны, мут, бан, кик, очистка чата |
| 🎟️ **Тикеты** | Система поддержки с несколькими ролями |
| 🎨 **Цветные роли** | Выбор цвета через реакции |
| 🎤 **Голосовые** | Приватные каналы с управлением |
| 🤖 **ИИ** | ChatGPT-подобный помощник |
| ⚧ **Гендер** | Роли Мужчина/Девушка |
